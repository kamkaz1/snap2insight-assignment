Tooth Paste Category Annotation Instructions

Precise and Succinct Guidelines for Annotation Team

Objective

Goal: Report as many tooth paste products as possible, even if variant/size is not visible. Tag them as "Tooth Paste Product".
  • Focus on identifying tooth paste products in retail images
  • Draw bounding boxes around all visible tooth paste items
  • Include all variants, sizes, and brands
  • Prioritize quantity over perfect precision

Product Identification Guidelines

What to Tag as "Tooth Paste Product":

  • All tooth paste brands: Colgate, Crest, Sensodyne, Aquafresh, etc.
  • All variants: Whitening, Sensitive, Fresh Mint, etc.
  • All sizes: Travel size, regular, family size
  • All packaging: Tubes, pumps, dispensers
  • Multi-packs: 2-pack, 3-pack, value packs
Examples:
  • Colgate Total Advanced Whitening
  • Crest Pro-Health Sensitive
  • Sensodyne Repair & Protect
  • Aquafresh Advanced Whitening

Handling Different Orientations

Angled Products

Rule: Draw bounding box around the entire visible product, even if it's at an angle.
  • Include the full product in the bounding box
  • Don't worry about perfect rectangular alignment
  • Focus on capturing the complete product

Turned Products

  • If you can see the product is tooth paste, tag it
  • Use context clues (shelf placement, nearby products)
  • When in doubt, include it

Occluded Products

Important: Only tag if at least 50% of the product is visible.
  • If partially hidden by other products, still tag if recognizable
  • If mostly hidden, skip it
  • Use your judgment based on visible portion

Single vs Multi-Pack Handling

Single Products

  • Draw one bounding box per individual product
  • Each tube/pump gets its own annotation
  • Tag as "Tooth Paste Product"

Multi-Packs

Rule: Draw bounding box around the entire multi-pack, not individual items.
  • 2-pack, 3-pack, value packs: one bounding box for the whole pack
  • Tag as "Tooth Paste Product"
  • Don't annotate individual items within the pack
Example: A 3-pack of Colgate should get one bounding box around the entire 3-pack, not three separate boxes.

Image Quality Issues

Blurry Images

  • If you can identify it as tooth paste, tag it
  • Use shape, size, and context clues
  • When in doubt, include it

Dark Images

  • Adjust your screen brightness if needed
  • Look for recognizable shapes and sizes
  • Use context from surrounding products

Angled Images

  • Draw bounding boxes that capture the full product
  • Don't worry about perfect alignment
  • Focus on completeness over precision

Non-Tooth Paste Products

Important Decision: Do NOT tag non-tooth paste products.

What NOT to Tag:

  • Tooth brushes
  • Mouthwash
  • Dental floss
  • Tooth whitening strips
  • Any other dental products
Focus: Only tooth paste products should be annotated. This keeps the dataset clean and focused.

Bounding Box Guidelines

Drawing Bounding Boxes:

  1. Click and drag to create rectangle around product
  2. Include the entire product within the box
  3. Don't worry about perfect alignment
  4. Ensure no part of the product is cut off

Quality Standards:

  • Box should encompass the full product
  • Minimal background included
  • Consistent approach across all images
  • When in doubt, make the box slightly larger

Quality Control Checklist

Before submitting each image:
  • ✓ All visible tooth paste products are tagged
  • ✓ No non-tooth paste products are tagged
  • ✓ Bounding boxes encompass full products
  • ✓ Multi-packs are treated as single units
  • ✓ Partially visible products (50%+) are included
  • ✓ All variants and sizes are captured
Remember: It's better to include a questionable product than to miss a clear tooth paste item.

Summary

Key Points:
  • Goal: Report maximum number of tooth paste products
  • Tag: All tooth paste variants, sizes, and brands
  • Don't Tag: Non-tooth paste products
  • Multi-packs: One bounding box per pack
  • Quality: Include if 50%+ visible
  • Approach: When in doubt, include it
Success Metric: High recall (finding all tooth paste products) is more important than perfect precision.
